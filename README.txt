CC other email addresses on emails bound to the admin email address.

== Description ==

This plugin allows you to add additional email address or select a user group to CC admin emails on.

Note, this will CC on ALL emails sent directly to the administrator.

== Installation ==

= Using The WordPress Dashboard =

1. Navigate to the 'Add New' in the plugins dashboard
2. Search for 'cc-admin-emails'
3. Click 'Install Now'
4. Activate the plugin on the Plugin dashboard

= Uploading in WordPress Dashboard =

1. Navigate to the 'Add New' in the plugins dashboard
2. Navigate to the 'Upload' area
3. Select `cc-admin-emails.zip` from your computer
4. Click 'Install Now'
5. Activate the plugin in the Plugin dashboard

= Using FTP =

1. Download `cc-admin-emails.zip`
2. Extract the `cc-admin-emails` directory to your computer
3. Upload the `cc-admin-emails` directory to the `/wp-content/plugins/` directory
4. Activate the plugin in the Plugin dashboard
